Models keep getting better. That doesn't close this gap: they still don't know how
*you* want code written, and they forget it the moment you switch tools. Linters
catch syntax. remindy catches **taste**: AI-slop copy, inline styles,
non-conventional commits, hardcoded secrets, and features nobody asked for. You teach
remindy once, and it holds every AI agent to those standards before it writes,
whatever model is behind it.
